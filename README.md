# MailForge

**MailForge** は、Thunderbirdのローカルメールフォルダーを監視し、新着メールの受信をリアルタイムで検知するElectronベースのデスクトップアプリケーションです。

## 概要

MailForgeは、Thunderbirdのメールストレージディレクトリへのファイル書き込みを監視し、メールの受信タイミングをログとして記録します。将来的には、受信メールの自動処理やメール送信の自動化などの機能拡張を予定しています。

## 主な機能

- **Thunderbirdメール監視**: Thunderbirdのローカルフォルダー（`Local Folders`）への書き込みを検知
- **リアルタイムログ**: 受信・更新イベントをタイムスタンプ付きでログ表示
- **監視の開始/停止**: UIから簡単に監視の開始・停止が可能
- **エラーハンドリング**: 監視中のエラーを適切に表示

## セットアップ

### 必要な環境
- Node.js (v14以上推奨)
- npm
- Thunderbird (インストール済み)

### インストール手順

1. リポジトリをクローンまたはダウンロード
2. 依存関係をインストール:
   ```bash
   npm install
   ```

3. アプリケーションを起動:
   ```bash
   npm start
   ```

## 使い方

1. アプリケーションを起動すると、メイン画面が表示されます
2. **「監視を開始」** ボタンをクリックして、Thunderbirdメールの監視を開始
3. Thunderbirdでメールを受信すると、ログエリアに受信イベントが表示されます
4. **「停止」** ボタンをクリックして監視を終了

## 技術スタック

- **Electron**: デスクトップアプリケーションフレームワーク
- **Node.js**: バックエンドロジック
- **chokidar**: ファイルシステム監視ライブラリ
- **HTML/CSS/JavaScript**: フロントエンドUI

## 開発者向け情報

### デバッグ

開発者ツールを開くには、アプリケーション起動後に `Ctrl+Shift+I` (Windows/Linux) または `Cmd+Option+I` (Mac) を押してください。

### Thunderbird監視の仕組み

`thunderbird.js` モジュールは、Thunderbirdのプロファイルディレクトリ内の `Mail/Local Folders` を監視します。このディレクトリへのファイル書き込みを検知することで、新着メールの受信を把握します。
